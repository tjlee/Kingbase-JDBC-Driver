# Kingbase JDBC Driver Changelog

## [1.0.0] - 2026-04-22

### Added
- KingbaseES database driver registration for DataGrip and all JetBrains IDEs with database support
- Bundled KingbaseES JDBC Driver versions 8.6.1 and 9.0.1 — no internet download required
- Native `jdbc:kingbase8://` URL scheme with host, port (default 54321), and database fields
